# Changelog

All notable changes to the **AOS Intelligence Dashboard** extension will be documented in this file.

## [0.1.0] - 2026-03-28

### Added
- Activity Bar icon and Sidebar Intelligence Dashboard with live GZMO metrics
- WebSocket-based TelemetryService with 10Hz throttling
- Benchmark Wizard panel with hardware routing override and AbortController support
- Model Leaderboard panel with in-memory sorting and demo data fallback
- Native Status Bar item with connection status
- State persistence across tab switches in all Webview panels
